# Starter

## Setup project

> jekyll new .

## Serve project

> bundle exec jekyll serve

* or

> jekyll serve

Browse project

> http://localhost:4000

## Use Chirpy

* Copy files

``` 
_data
_plugins
_tabs
_config.yml
index.html
```

* Remove index.markdown and about.markdown
* Run `bundle`

## Create new post

> jekyll post <file-name>
